Welcome!

To run locally:

- Install npm and node.js
- In hydrozoa-labs:
    - Run `npm install`
    - Run `npm run build` to build the react app
- In hydrozoa-labs-backend
    - Run `npm install`
    - Run `node app.js` to launch the server
- Visit localhost:3000 in a web browser, and you should see the site!
